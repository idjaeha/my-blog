# Project Instructions

## Plan Documentation

새로운 작업 계획을 세울 때 `docs/plans/` 디렉토리에 문서를 작성한다.

- 파일명: `YYYY-MM-DD-간결한-설명.md`
- 포맷: `docs/plans/_template.md`를 따른다

## Troubleshooting Documentation

버그를 해결한 후에는 `docs/troubleshooting/` 디렉토리에 문서를 작성한다.

- 파일명: `NNN-간결한-설명.md` (번호는 기존 파일의 다음 순번)
- 포맷: `docs/troubleshooting/_template.md`를 따른다
