## 안녕하세요

이 블로그는 **Astro**와 **MDX**를 기반으로 만들어졌습니다.

<Callout type="tip" title="MDX 지원">
  이 블로그는 MDX를 지원하여 마크다운 안에서 React 컴포넌트를 사용할 수
  있습니다.
</Callout>

### 기술 스택

- **Astro** — 정적 사이트 생성기
- **React** — 인터랙티브 아일랜드 컴포넌트
- **Tailwind CSS** — 유틸리티 우선 스타일링
- **shadcn/ui** — 접근 가능한 UI 컴포넌트
