블로그 글을 작성합니다.

먼저 `docs/guides/writing-post.md`를 읽고 작성 규칙을 확인한 후, 아래 절차를 따릅니다.

1. 주제에 맞는 slug, title, description, category, tags를 결정한다
2. `src/content/blog/ko/[slug].mdx` 파일을 생성한다
3. 프론트매터와 본문을 작성한다
4. 필요한 MDX 컴포넌트(Callout, LinkCard, MermaidDiagram 등)를 활용한다
5. `pnpm build`로 빌드 에러가 없는지 확인한다
